# PWM Generator

Простой проект для управления сервоприводом с помощью STM32 Blue Pill (STM32F103C8T6).

## Описание

Проект циклически управляет сервоприводом, переключая его между тремя позициями:
- Минимальная позиция (1000 мкс)
- Центральная позиция (1500 мкс) 
- Максимальная позиция (2000 мкс)

Каждая позиция удерживается в течение 5 секунд.

## Оборудование

- STM32 Blue Pill (STM32F103C8T6)
- Сервопривод
- ST-Link для программирования

## Подключение

Сервопривод подключается к пину PA8 микроконтроллера.

## Сборка и загрузка

Проект использует PlatformIO для сборки и загрузки:

```bash
pio run -t upload
```
