# План разработки ROV

## Полтава

### 1. Проводной пульт управления
Эргономически удобное печатное изделие, стилизованное под аппаратуру радиоуправления FlySky

- **Органы управления**: два аналоговых стика и 6 кнопок, расположенные парами в различных зонах
- **Кабель**: 3 метра c разьемами с двух сторон. 
- **Электроника**: встроенная плата, преобразующая сигналы от стиков и кнопок в строку данных для передачи на робота по протоколу RS485

### 2. Пост управления на панели
Компактная компоновка компонентов управления на пластине с защитой от возможных протечек.
- **Экран**: размещен на регулируемом кронштейне с возможностью изменения угла наклона
- **Зона для пульта**: специально отведенное место под экраном для размещения пульта управления

### 3. Комплектация роботов
- **Общее количество**: 4 комплекта
- **Рабочие**: 2 комплекта
- **Запасные**: 2 комплекта


## Орленок (февраль 2025)

### 1. Проводной пульт управления
Использование проверенного пульта с проекта "Полтава"

### 2. Пост управления
- **Ниша для пульта**: специальное углубление с кабельным каналом
- **Защитная пластина**: закрывает техническую часть за экраном

### 3. Программируемый драйвер моторов *(в исследовании)*
Разработка собственного драйвера для предотвращения проблем с моторами и снижения стоимости за счет покупки моторов без встроенных драйверов

### 4. Комплекты печатных плат
- **Тестовые**: 2 комплекта
- **Финальные**: 20 комплектов
- **Состав**: плата для пульта + плата для робота с полной гальванической развязкой

### 5. Герметичные корпуса (колбы)
- **Тестовая партия**: 1 штука
- **Финальная партия**: 20 штук
- **Разъем**: SubConn 20-pin в крышке
- **Возможные доработки**: 
  - Увеличение размера колбы для размещения драйвера
  - Интеграция камеры с акриловым куполом (как у Proteus) для упрощения пластиковой рамы

### 6. Манипулятор *(в исследовании)*
При наличии к концу года стабильной версии самодельного манипулятора:
- **Корпус**: точеный алюминиевый герметичный
- **Губки**: фрезерованные или напечатанные пластиковые
- **Гермоввод**: с резиновой заглушкой для снижения стоимости

### 7. Гермовводы
Возможный заказ на изготовление идентичных разработанным образцам с доработками:
- Канавка внутри для улучшения фиксации гермокаста

### 8. Пластиковая рама
Изготовление под новые технические требования, которые будут определены к концу года:
- **Особенность**: без вплавляемых закладных элементов

Пульт управления:
1) Отказ от джойстика PS4 
2) Отказ от одноплатного компьютера Radxa 
3) Модернизация платы поста управления на микроконтроллер семейства STM32
4) Меняем интерфейс с RS485 (полудуплекс) на гальванически развязанный RS422 (дуплекс)

Робот:
1) Переезжаем на драйвера BLDC моторов в блоке электроники с возможностью гибкой настройки 
2) Меняем интерфейс с RS485 (полудуплекс) на гальванически развязанный RS422 (дуплекс)
3) Переезжаем на микроконтроллер семейства STM32 
4) Модернизация платы робота
5) Интеграция СУ с обратной связью по датчику глубины и IMU 
6) Гальваническая развязка логической части от силовой
